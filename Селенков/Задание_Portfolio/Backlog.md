# Product Backlog: Portfolio Manager

## Эпики и пользовательские истории

### Эпик 1: Управление записями портфолио
**Приоритет: Высокий**
**Оценка: 40 story points**

#### US-001: Создание новой записи
**Как** пользователь системы,
**Я хочу** создавать новые записи портфолио,
**Чтобы** сохранять информацию о своих достижениях.

**Критерии приемки:**
- Можно ввести название, тип, год
- Автоматически создается Markdown файл
- Запись появляется в списке
- Сохраняется в базе данных

#### US-002: Редактирование существующей записи
**Как** пользователь системы,
**Я хочу** редактировать записи портфолио,
**Чтобы** обновлять информацию о достижениях.

#### US-003: Удаление записей
**Как** пользователь системы,
**Я хочу** удалять записи портфолио,
**Чтобы** удалять устаревшую или ненужную информацию.
